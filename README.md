# OpenWrt BPF CO-RE

This repository provides pre-generated `vmlinux.btf` files for the kernels of **official OpenWrt releases**, aiming to assist developers in running **BPF CO-RE (Compile Once - Run Everywhere)** programs on official OpenWrt firmware that **do not have kernel BTF support enabled**.

-----

Currently supported platforms:
- x86/64
- rockchip/armv8
